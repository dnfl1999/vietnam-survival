# Vietnam Survival for Koreans 인수인계서

작성일: 2026-04-08

## 1. 오늘 한 작업 요약

베트남 거주자/여행 중인 한국인을 위한 iPhone 전용 SwiftUI MVP 앱 초안을 작성했다.

앱 목적:
- 번역 앱이 아니라 "상황 해결 버튼 앱"
- 병원, 택시, 음식 주문, 흥정, 위험 상황 등에서 버튼만 눌러 바로 문장을 보여주고 음성으로 재생하는 구조

오늘 만든 산출물은 두 버전이다.

1. `VietnamSurvivalForKoreans`
- 첫 MVP 버전
- 기본 구조, 샘플 데이터, TTS, 즐겨찾기, 검색, 프리미엄 placeholder 포함

2. `VietnamSurvivalForKoreans_v2`
- UI 개선 버전
- 홈 화면 가독성 강화
- 빠른 진입 버튼 추가
- 카드 디자인 통일
- 위험 상황 강조 강화
- 전체 화면 보여주기 모드 개선
- 프리미엄 placeholder 확장

실질적으로 다음 작업은 `VietnamSurvivalForKoreans_v2` 기준으로 이어가면 된다.

## 6. 아직 안 한 것 / 주의사항

중요:
- 현재는 `소스 파일만 정리된 상태`다.
- `Xcode 프로젝트(.xcodeproj)`는 아직 만들지 않았다.
- 이 작업 환경은 Windows라서 iPhone 앱 실행/빌드는 하지 못했다.
- 실제 실행은 Mac + Xcode가 필요하다.

즉, 내일 이어서 할 때 우선순위는 아래와 같다.

1. Mac 환경에서 Xcode 프로젝트 생성
2. `VietnamSurvivalForKoreans_v2` 파일을 프로젝트에 추가
3. iOS 17 시뮬레이터에서 빌드 확인
4. 컴파일 에러/리소스 연결 문제 수정
5. 실제 UI 미세 조정

## 7. 예상 체크 포인트

내일 Mac에서 확인해야 할 항목:

- `@main` 앱 파일이 정상 진입하는지
- 모든 Swift 파일이 타깃에 포함됐는지
- `AVFoundation` TTS가 `vi-VN` 음성으로 재생되는지
- `UserDefaults` 즐겨찾기가 정상 저장되는지
- `searchable`가 실시간 검색처럼 잘 반응하는지
- `fullScreenCover` 전체 화면 모드가 정상 표시되는지
- 다크모드 가독성
- 작은 화면(iPhone SE 계열)에서 버튼 겹침 없는지

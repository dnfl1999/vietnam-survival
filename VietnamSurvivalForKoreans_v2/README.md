# Vietnam Survival for Koreans v2

UI 개선 버전입니다.

## 개선 내용

- 홈 상단 소개 문구 추가
- 홈 하단 빠른 진입 버튼 추가
- 카테고리 카드 스타일 통일
- 위험 상황 카드/문장 강조 강화
- 문장 카드에서 베트남어 크게 표시
- 성조 없는 베트남어를 보조 텍스트로 축소
- 카드 하단에 즐겨찾기와 음성 버튼 고정 배치
- 문장 상세 화면의 전체 화면 보여주기 모드 강화
- 검색은 `searchable` 바인딩 기반 실시간 검색
- 프리미엄 Placeholder 항목 확장

## 폴더 구조

```text
VietnamSurvivalForKoreans_v2/
├─ App/
├─ Data/
├─ Managers/
├─ Models/
├─ ViewModels/
└─ Views/
   └─ Components/
```

## 실행 방법

1. Xcode에서 iOS App 프로젝트를 생성합니다.
2. 배포 타깃을 iOS 17 이상으로 설정합니다.
3. 이 폴더 안의 파일들을 프로젝트에 추가합니다.
4. 기본 샘플 파일은 제거하고 실행합니다.

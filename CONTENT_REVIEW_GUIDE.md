# 콘텐츠 검수 가이드

작성일: 2026-04-10

검수 대상 CSV: `review/phrases_review_2026-04-10.csv`

## 목표

앱에 들어간 120개 문장이 실제 베트남 현지에서 보여주거나 들려줘도 자연스럽고 안전한지 확인한다.

## 검수 우선순위

1. `emergency`
   - 오역이 있으면 실제 위험 상황에서 문제가 될 수 있으므로 최우선 검수
   - 경찰, 구급차, 대사관, 도난, 따라오지 말라는 문장은 의미가 강하므로 표현 강도 확인

2. `hospitalPharmacy`
   - 증상, 약, 복용법 관련 문장은 의미가 정확해야 함
   - 의학적으로 과도하게 단정적인 표현이 없는지 확인

3. `taxiGrab`
   - 기사에게 바로 보여줄 때 자연스러운지 확인
   - 그랩/택시 상황 모두에 어색하지 않은지 확인

4. `foodOrder`
   - 알레르기, 돼지고기, 땅콩, 해산물 표현 우선 확인
   - 식당에서 실제로 통하는 자연스러운 주문 표현인지 확인

5. `priceNegotiation`
   - 현지 상점/시장 상황에서 너무 무례하지 않은지 확인
   - 환불/교환 표현이 자연스러운지 확인

6. `commonPhrases`
   - 기본 표현이므로 자연스러움 중심으로 확인

## CSV 컬럼 설명

- `id`: 앱 내부 문장 ID
- `category`: 앱 카테고리
- `korean`: 한국어 원문
- `vietnamese`: 성조 포함 베트남어
- `vietnamesePlain`: 급할 때 따라 읽기 쉬운 무성조 표기
- `isEmergency`: 위험 상황 강조 여부
- `memo`: 검수자가 수정 의견을 적는 칸

## 검수자에게 부탁할 질문

- 한국어 뜻과 베트남어 뜻이 정확히 맞나요?
- 실제 베트남 사람이 보기에 자연스러운 표현인가요?
- 더 짧고 바로 통하는 표현이 있나요?
- 응급/도난/경찰 관련 표현이 너무 약하거나 너무 강하지 않나요?
- TTS로 읽혔을 때 어색할 가능성이 큰 문장이 있나요?
- 성조 없는 표기(`vietnamesePlain`)가 따라 읽기용으로 충분히 맞나요?

## 수정 반영 규칙

- 검수자는 원문을 직접 덮어쓰기보다 `memo`에 수정안을 적는다.
- 수정 반영 시 `vietnamese`와 `vietnamesePlain`을 함께 업데이트한다.
- 응급 문장은 수정 후 한 번 더 재확인한다.
- 의미가 애매하면 앱에 넣기보다 보류한다.

## 보류 추천 사례

- 법률/의료적으로 책임이 생길 수 있는 표현
- 현지에서 무례하게 받아들여질 수 있는 표현
- 한국어 뜻은 좋아도 베트남어가 너무 길어 긴급 상황에서 쓰기 어려운 표현
- 특정 도시나 서비스에만 맞는 표현

# 제품 백로그

작성일: 2026-04-10

기준 앱: `VietnamSurvivalForKoreans_v2`

## 현재 MVP 범위

- SwiftUI iOS 앱 소스
- 6개 카테고리
- 문장 120개
- 검색
- 즐겨찾기 저장
- 베트남어 TTS
- 문장 크게 보여주기
- 프리미엄 placeholder

## P0: Mac/Xcode 첫 실행 전후

- Xcode 프로젝트 생성
- `VietnamSurvivalForKoreans_v2` Swift 파일 전체 타깃 추가
- 첫 빌드 오류 수정
- iPhone Simulator 첫 실행
- `@main` 진입점 중복 여부 확인
- iPhone SE 계열 레이아웃 확인
- TTS `vi-VN` 음성 출력 확인
- 즐겨찾기 저장/복원 확인
- 검색에서 한국어, 성조 포함 베트남어, 무성조 베트남어 모두 확인

## P1: 콘텐츠 신뢰도

- `review/phrases_review_2026-04-10.csv` 베트남어 검수
- 응급 문장 20개 우선 검수
- 병원/약국 문장 20개 우선 검수
- 검수 반영 후 `SamplePhrases.swift` 업데이트
- 앱 내 문장 수와 CSV 문장 수 일치 확인

## P1: UX 개선

- 홈 빠른 진입에서 중첩 `NavigationStack`이 어색한지 확인
- 문장 카드 하단 버튼이 작은 화면에서 답답한지 확인
- 긴 베트남어 문장의 줄바꿈 확인
- 전체 화면 모드에서 글자가 너무 작거나 큰지 확인
- 검색 결과가 많을 때 카테고리 표시가 충분한지 확인
- 응급 문장을 홈에서 더 빠르게 접근할 수 있게 할지 결정

  - 경찰/분실
  - 공항/입국

  - 유심/통신
- 문장 데이터 JSON 분리

- 즐겨찾기 내 검색
- 카테고리별 문장 정렬 커스터마이즈

## 의사결정 메모

- 지금은 기능을 늘리기보다 첫 실행과 콘텐츠 신뢰도가 우선이다.
- Mac/Xcode 전에는 데이터, 문서, 검수 준비를 끝내는 것이 가장 효율적이다.
- 응급/병원 문장은 전문 검수 전까지 “초안”으로 취급한다.
